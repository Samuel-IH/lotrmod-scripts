# lotrmod-scripts

Tiny repo for scripts that tie together a LOTR modpack (1.7.10). The included mods are:

- Treecapitator
- Applecore
- Chisel
- Cookingbook
- Crafttweaker (duh)
- Eb's wizardry
- Fastleafdecay
- journeymap (for now)
- LOTRMod
- Nei+plugins
- Nomadic tents
- Squeedometer (for horse bragging rights!)
- Waystones (idk why)
- Worldedit
